# 苹果 (AAPL) 企业动态日报

**日期**: 2026年2月28日 (周六)

## 🔍 今日要闻

### iOS 与软件
- **iOS 26.4 Beta 年龄验证提示争议**: 苹果确认英国用户在下载 iOS 26.4 Beta 时看到的年龄验证提示是"错误"，现已修复。发言人 Peter Ajemian 表示开发者可继续使用 Declared Age Range API 为用户提供适龄体验。

### 产品动态
- **新维修政策**: 苹果持续扩展自助维修计划覆盖范围
- **Apple Intelligence**: AI 功能持续扩展至更多设备和市场

### 服务更新
- **年龄验证 API**: 苹果为开发者提供年龄范围声明 API，支持适龄内容筛选
