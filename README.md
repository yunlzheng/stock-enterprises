# 📊 Stock Enterprises - 企业动态追踪

每日自动追踪关注公司的最新新闻、财报和市场动态。

## 📁 目录结构

```
stock-enterprises/
├── companies/           # 公司动态数据
│   ├── NVIDIA/         # 英伟达
│   ├── BABA/           # 阿里巴巴
│   ├── MSFT/           # 微软
│   ├── AAPL/           # 苹果
│   └── TSLA/           # 特斯拉
├── templates/          # 模板文件
└── config.json         # 配置文件
```

## 🔔 定时任务

每天早上 9:00 (GMT+8) 自动：
1. 通过浏览器获取各公司最新动态
2. 推送到飞书
3. 提交数据到本仓库

## 📝 数据格式

每个公司的日报包含：
- 📰 重要新闻
- 📊 股价变动
- 📈 财报要点
- 🔮 市场预期
